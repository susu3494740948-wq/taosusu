## 平台差异对照表

| 维度 | TikTok Shop | Shopee | Temu |
|------|-------------|--------|------|
| 内容形式 | 短视频 hook 必填，9:16 封面 | 图文为主，可加短视频 | 供货图 + 质检图为主 |
| 标题长度 | ≤ 80 字符（英文） | 120 字符内，可双语 | 由平台生成，供货端填属性 |
| 定价模式 | 商家定价 + 平台佣金 | 商家定价 + 店铺优惠券 | 供货价 + 平台定前台价 |
| 物流 | 商家自发货或平台物流 | 运费模板 + 跨境/本地仓 | 平台揽收 / 半托管 |
| 关键字段 | 合规声明、视频素材 | 类目路径、变体、运费 | 供货价、包装清单、重量尺寸 |
| 助理核心动作 | 标题/bullet/视频脚本整理 | 类目+运费+多语言标题 | 供货价、质检图、包装清单 |

## 新品上架 Checklist（15 步）

1. [ ] 确认内部 SKU 编号与产品名称
2. [ ] 收集供货价、MOQ、重量、尺寸
3. [ ] 确认合规文件状态（电池/液体/儿童等）
4. [ ] 拍摄/收集主图（白底 + 场景 + 对比）
5. [ ] 撰写英文标题（含核心关键词）
6. [ ] 撰写 5 条 bullet points
7. [ ] 选择各平台类目路径
8. [ ] 填写定价与运费模板
9. [ ] 准备 TikTok 9:16 短视频脚本
