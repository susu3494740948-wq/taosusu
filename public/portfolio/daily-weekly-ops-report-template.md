## 一、日报模板

### 字段说明

| 字段 | 说明 |
|------|------|
| 日期 | YYYY-MM-DD |
| 平台 | TikTok Shop / Shopee / Temu / 独立站 |
| 曝光 | Impressions |
| 点击 | Clicks |
| 加购 | Add to carts |
| 订单 | Orders |
| GMV | 成交总额（USD） |
| 退款 | 退款笔数或金额 |
| 备注/异常 | 当天需跟进事项 |

### 空白模板（复制使用）

```csv
日期,平台,曝光,点击,加购,订单,GMV_USD,退款,备注
```

## 四、SOP 摘要

### 订单异常处理流程

```
收到异常 → 分类（缺货/地址错误/物流超时/质量问题）
  → 缺货：查库存 + 通知主管 + 联系客户改发或退款
  → 地址错误：24h 内联系客户确认
  → 物流超时：查追踪号 + 按平台 SLA 回复
  → 质量问题：escalation 主管 + 记录批次号
```
